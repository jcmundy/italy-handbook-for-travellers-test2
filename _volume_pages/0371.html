{% raw %}Adjoining the church of S. Giovanni in Laterano, on the N.{% endraw %}
{% raw %}side, is the —{% endraw %}
{% raw %}Palazzo del Laterano (PL II, 30), to which, together with the{% endraw %}
{% raw %}Vatican (p. 288) and Castel Gandolfo, the privilege of exterritori¬{% endraw %}
{% raw %}ality was secured by a law of 13th May, 1871. This was the residence{% endraw %}
{% raw %}of the popes from the time of Constantine down to the migration to{% endraw %}
{% raw %}Avignon. The old palace was much more extensive than the present,{% endraw %}
{% raw %}and included also the Sancta Sanctorum Chapel (p. 268). After a{% endraw %}
{% raw %}great fire in 1308 it lay in ruins, which were removed by order of{% endraw %}
{% raw %}Sixtus V. and the newpalace erected by Domenico Fontana in 1586.{% endraw %}
{% raw %}As it remained unoccupied, it was converted by Innocent XII. into{% endraw %}
{% raw %}an orphan asylum in 1693. In 1843 Gregory XVI. set apart the{% endraw %}
{% raw %}palace for a collection of the heathen and Christian antiquities for{% endraw %}
{% raw %}which the Vatican and Capitoline museums no longer afforded space,{% endraw %}
{% raw %}and named it the 'Museum Gregorianum Lateranense, a collection{% endraw %}
{% raw %}which has since then steadily increased in importance. Visitors{% endraw %}
{% raw %}admitted daily, except on holidays, 9-3 o'clock. The entrance{% endraw %}
{% raw %}is by ihe portal in the piazza opposite the obelisk (p. 267); visitors{% endraw %}
