{% raw %}Weapons, which for the ordinary traveller are a mere burden,{% endraw %}

{% raw %}to confiscation.{% endraw %}
{% raw %}Begging, which was countenanced and even encouraged under{% endraw %}
{% raw %}the old system of Italian politics, still continues to be one of those{% endraw %}
{% raw %}national nuisances to which the traveller must habituate himself.{% endraw %}
{% raw %}The present government has adopted energetic measures for its sup¬{% endraw %}
{% raw %}pression, but hitherto with only partial success. The average Italian{% endraw %}
{% raw %}beggar is a mere speculator, and not a deserving object of charity.{% endraw %}
{% raw %}The traveller should therefore decline to give anything, with the{% endraw %}
{% raw %}words, 'non c'e niente', or a gesture of disapproval. If a donation{% endraw %}
{% raw %}be bestowed, it should consist of one of the smallest possible copper{% endraw %}
{% raw %}coins. A beggar, who on one occasion was presented with 2c. and{% endraw %}
{% raw %}thanked the donor with the usual benedictions, was on another pre¬{% endraw %}
{% raw %}sented with 50c; but this act of liberality, instead of being grate¬{% endraw %}
{% raw %}fully accepted, only called forth the remark in a half-offended tone :{% endraw %}
{% raw %}— 'Ma, signore, e molto poco!'{% endraw %}
{% raw %}VI. Intercourse with Italians.{% endraw %}
{% raw %}In Italy the pernicious custom of demanding considerably more{% endraw %}
{% raw %}than will ultimately be accepted has long been prevalent; but{% endraw %}
{% raw %}a knowledge of the custom, which is based on the presumed igno¬{% endraw %}
{% raw %}rance of one of the contracting parties, tends greatly to mitigate the{% endraw %}
{% raw %}evil. Where tariffs and fixed charges exist, they should be carefully{% endraw %}
{% raw %}consulted. In other cases, where a certain average price is establish¬{% endraw %}
{% raw %}ed by custom, the traveller should make a distinct bargain as to{% endraw %}
{% raw %}the article to be bought or service to be rendered, and never rely on{% endraw %}
{% raw %}the equity of the other party. Nor should any weight be attached to{% endraw %}
{% raw %}the representations of waiters, drivers, and guides in matters in{% endraw %}
{% raw %}which they have an interest, and with whom even the inhabitants{% endraw %}
{% raw %}of the place often appear to act in concert.{% endraw %}
