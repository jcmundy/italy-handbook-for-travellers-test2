{% raw %}Excursions (S. E.). VIA APPIA. Environs of Rome. 345{% endraw %}
{% raw %}On a marble tablet facing the road is inscribed : Caeciliae Q. Cretici{% endraw %}
{% raw %}Filiae Metellae Crassi, i.e. to the manes of the daughter of Metellus{% endraw %}
{% raw %}Creticus, wife of the triumvir Crassus. The interior, now almost{% endraw %}
{% raw %}entirely filled up, contained the tomb-chamber of the deceased. In{% endraw %}
{% raw %}the 13th cent, the Gaetani converted the edifice into the tower of{% endraw %}
{% raw %}a stronghold, and furnished it with pinnacles. To this extensive{% endraw %}
{% raw %}castle , which subsequently passed through various hands, and was{% endraw %}
{% raw %}destroyed under Sixtus V., belong the picturesque ruins of a palace{% endraw %}
{% raw %}adjacent to the tower, and a church opposite.{% endraw %}
{% raw %}As far as this neighbourhood extends a lava-stream which once{% endraw %}
{% raw %}descended from the Alban Mts. and yielded paving material for the{% endraw %}
{% raw %}ancient road. The more interesting part of the Via now begins ; the{% endraw %}
{% raw %}ancient pavement is visible in many places, the road is skirted on{% endraw %}
{% raw %}both sides by continuous rows of ruined tombs, and the view be¬{% endraw %}
{% raw %}comes more extensive at every step. On the left are perceived the{% endraw %}
{% raw %}adjacent arches of the Aqua Marcia and the Aqua Claudia, the latter{% endraw %}
{% raw %}now partly converted into the modern Acqua Felice (comp. p. 348).{% endraw %}
{% raw %}The houses on the road-side gradually cease, and 2*/4 M. from the{% endraw %}
{% raw %}city-gate we reach the entrance (indicated by a notice on a house to{% endraw %}
{% raw %}the right) to the excavated part of the Via Appia , flanked beyond{% endraw %}
{% raw %}this point by a constant succession of tombs. Many of these contain{% endraw %}
{% raw %}reliefs and inscriptions worthy of note. The scenery continues to{% endraw %}
{% raw %}be strikingly beautiful.{% endraw %}
{% raw %}On the left, IL/4M. from the entrance, is a 'casale' built within the{% endraw %}
{% raw %}walls of an ancient church, which is called <S. Maria Nuova. Beyond{% endraw %}
{% raw %}it lie the extensive ruins named Roma Vecchia, which appear to{% endraw %}
{% raw %}have belonged to a spacious villa of the Quintilii. Several of the{% endraw %}
{% raw %}chambers were employed as baths.{% endraw %}
{% raw %}A large tomb on the left, the site of which is now occupied by{% endraw %}
{% raw %}a small farm, 3/4 M. from S. Maria Nuova, is named the Casale{% endraw %}
{% raw %}Rotondo. It lies by the 6th milestone, and, according to Canina,{% endraw %}
{% raw %}was erected for Messala Corvinus, a statesman and poet of consid¬{% endraw %}
{% raw %}erable reputation under Augustus, but this conjecture is not sup¬{% endraw %}
{% raw %}ported by sufficient evidence. It may be ascended for the sake of{% endraw %}
{% raw %}the fine view it commands (30(c). — The lofty building on the left,{% endraw %}
{% raw %}7 min. farther, on the same side, is also an ancient tomb on which{% endraw %}
{% raw %}the Arabs and Normans erected a tower, named Tor di Selce (tower{% endraw %}
{% raw %}of basalt).{% endraw %}
{% raw %}The prolongation of the Via Appia from this point to Albano{% endraw %}
{% raw %}(7'/a M.) is less interesting. After 1 M. a field-road diverges to the{% endraw %}
{% raw %}left, leading to the Via Appia Nuova (see below). Among the tombs{% endraw %}
{% raw %}may also be mentioned, on the left, 2 M. beyond Tor di Selce, the{% endraw %}
{% raw %}circular Torraccio, or Palombaro, the name of whose occupant is{% endraw %}
{% raw %}unknown. At the 11th milestone the road is crossed by the railway,{% endraw %}
{% raw %}a little beyond which is the Osteria delle Fratocchie; thence to{% endraw %}
{% raw %}Albano, see p. 359.{% endraw %}
