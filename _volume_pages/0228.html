{% raw %}* The Corso.{% endraw %}
{% raw %}The *Corso, which corresponds with the ancient Via Flaminia{% endraw %}
{% raw %}leading from the Capitol, extends from the Piazza del Popolo{% endraw %}
{% raw %}(p. 140), between the Via di Ripetta and Via del Babuino, to the{% endraw %}
{% raw %}Piazza di Venezia, and is now the principal street of Rome, with{% endraw %}
{% raw %}numerous shops, and enlivened, especially towards evening, by{% endraw %}
{% raw %}crowds of carriages and foot-passengers. The Carnival is celebrated{% endraw %}
{% raw %}here, and the whole street is then thickly strewn with puzzolana{% endraw %}
{% raw %}earth for the horse-races , the horses starting from the Piazza del{% endraw %}
{% raw %}Popolo. From the Piazza del Popolo to the Via Condotti is a distance{% endraw %}
