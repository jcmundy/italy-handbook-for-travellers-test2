{% raw %}Near the above-named church of S. Giorgio is an iron gate{% endraw %}
{% raw %}(attendant 4-5 soldi), leading to the Grotto of Neptune, which was{% endraw %}
{% raw %}formerly the channel of the main branch of the Anio. The new{% endraw %}
{% raw %}works drew off the greater part of the water from this channel, but{% endraw %}
{% raw %}the fall is still very fine. The excellent path, affording picturesque{% endraw %}
{% raw %}glimpses of the great fall, was constructed by the French general{% endraw %}
{% raw %}Miollis. Following this path to the right, we cross a wooden bridge,{% endraw %}
{% raw %}and reach the grotto in a few minutes. — We now return to the gal¬{% endraw %}
{% raw %}lery hewn in the rock, and pass through it, descend to the lowest{% endraw %}
{% raw %}point to which the path leads, and finally mount a flight of stone{% endraw %}
{% raw %}steps, constantly wetted by spray, to the fantastically shaped{% endraw %}
{% raw %}■"Sirens' Grotto. From this point we return to the path, which{% endraw %}
{% raw %}ascends at first in steps and afterwards in zigzags. Halfway up,{% endraw %}
{% raw %}where the zigzags terminate, near some cypresses, a path diverges{% endraw %}
{% raw %}to the left, at first ascending a few steps, then descending, and{% endraw %}
{% raw %}leading in a few minutes to a Platform of masonry, immediately{% endraw %}
{% raw %}above the new fall. — We now return and continue to ascend, and{% endraw %}
{% raw %}at the top of the hill turn to the left to a Terrace planted with{% endraw %}
{% raw %}olives, whence we enjoy a charming * view of the temple of the{% endraw %}
{% raw %}Sibyl, above us, and of the new waterfall below. Passing through{% endraw %}
{% raw %}a door, which a custodian opens (fee), we may proceed to the fall{% endraw %}
{% raw %}and the entrance of the two tunnels by which the Monte Catillo is{% endraw %}
{% raw %}penetrated (see above; 372 steps long; to persons inclined to giddi¬{% endraw %}
{% raw %}ness the walk by the roaring stream is not recommended). — From{% endraw %}
{% raw %}the terrace we follow the main path on the same level, constantly{% endraw %}
{% raw %}affording a view of the temples on the right. The path then passes{% endraw %}
{% raw %}under the street and reaches the exit to the right (10c). — Visitors{% endraw %}
{% raw %}now generally walk through the town gate (Porta S. Angelo) to the{% endraw %}
{% raw %}right, and beyond it follow the road to the left, which runs at the{% endraw %}
{% raw %}same level along the slope above the river, and affords a series of{% endraw %}
{% raw %}more extensive views of the falls. Fine olive trees on the road.{% endraw %}
{% raw %}After 3/4 M. we reach the first point of view, a round Terrace en¬{% endraw %}
{% raw %}closed by masonry, and ('/2 M.) a second *Terrace, commanding an{% endraw %}
{% raw %}admirable view of the new fall, of Tivoli, and of the smaller water¬{% endraw %}
{% raw %}falls below the town, known as *Le Cascatelle, formed by a branch{% endraw %}
{% raw %}of tlie Anio.{% endraw %}
