{% raw %}PREFACE.{% endraw %}
{% raw %}The Maps and Plans, on which special care has been{% endraw %}
{% raw %}bestowed, will abundantly suffice for the use of the ordinary{% endraw %}
{% raw %}traveller. The Plan of Rome (scale 1 :15,000) is divided{% endraw %}
{% raw %}into three sections with a view to obviate the necessity of{% endraw %}
{% raw %}unfolding a large sheet of paper at every consultation, and{% endraw %}
{% raw %}its use will be further facilitated by reference to the small{% endraw %}
{% raw %}clue-plan inside the cover at the end of the volume.{% endraw %}
{% raw %}Heights are given in English feet (1 Engl. ft. = 0,3048{% endraw %}
{% raw %}metre), and Distances in English miles (comp. p. ii).{% endraw %}
{% raw %}Hotels (comp. p. xxi). In no country does the treat¬{% endraw %}
{% raw %}ment which the traveller experiences at hotels vary more{% endraw %}
{% raw %}than in Italy, and attempts at extortion are perhaps no¬{% endraw %}
{% raw %}where so outrageous. Besides the modern palatial and ex¬{% endraw %}
{% raw %}pensive establishments, the Handbook also contains a selec¬{% endraw %}
{% raw %}tion of modest, old-fashioned inns, where gentlemen trav¬{% endraw %}
{% raw %}elling alone will not unfrequently find good accommodation{% endraw %}
{% raw %}at moderate charges. The asterisks indicate those hotels{% endraw %}
{% raw %}which the Editor has reason to believe from his own ex¬{% endraw %}
{% raw %}perience, as well as from information supplied by numerous{% endraw %}
{% raw %}travellers, to be respectable, clean, and reasonable. The{% endraw %}
{% raw %}value of these asterisks, it need hardly be observed, varies{% endraw %}
{% raw %}according to circumstances, those prefixed to town hotels{% endraw %}
{% raw %}and village inns signifying respectively that the establish¬{% endraw %}
{% raw %}ments are good of their kind. At the same time the Editor{% endraw %}
{% raw %}does not doubt that comfortable quarters may occasionally{% endraw %}
{% raw %}be obtained at inns which he has not recommended or even{% endraw %}
{% raw %}mentioned. Although changes frequently take place, and{% endraw %}
{% raw %}prices generally have a strong upward tendency, the average{% endraw %}
{% raw %}charges stated in the Handbook will enable the traveller to{% endraw %}
{% raw %}form a fair estimate of his probable expenditure.{% endraw %}
{% raw %}To hotel proprietors, tradesmen, and others the Editor{% endraw %}
{% raw %}begs to intimate that a character for fair dealing and cour¬{% endraw %}
{% raw %}tesy towards travellers forms the sole passport to his com¬{% endraw %}
{% raw %}mendation, and that advertisements of every kind are strictly{% endraw %}
{% raw %}excluded.{% endraw %}
