{% raw %}The Vatican. ROME. V. Right Bank. 289{% endraw %}
{% raw %}The Principal Entrance, to the Vatican{Portone di Bronzo) is{% endraw %}
{% raw %}at the end of the right colonnade of the Piazza of St. Peter. Within{% endraw %}
{% raw %}the gate visitors formerly proceeded, immediately beyond the Swiss{% endraw %}
{% raw %}guard, to a flight of steps to the right, by which however they are{% endraw %}
{% raw %}now taken hya Swiss to the Maggiordomato only (p. 118) to receive{% endraw %}
{% raw %}their permesso. (Present entrance to the Stanze, etc., see below.){% endraw %}
{% raw %}The steps, originally uncovered, were altered by Pius IX., and lead{% endraw %}
{% raw %}to the Cortile di S. Damaso, a court which derives its name from{% endraw %}
{% raw %}the fountain of St. Damasus erected here by Innocent X., and some¬{% endraw %}
{% raw %}times called Cortile delle Loggie from the Loggie of Bramante (p.{% endraw %}
{% raw %}302) by which it is bounded on three sides. On the right is the{% endraw %}
{% raw %}wing occupied by the Pope; on the left a door with the inscription{% endraw %}
{% raw %}Adito alia Biblioteca ed al Museo (now available for workers in the{% endraw %}
{% raw %}library only) leads to the staircase which ascends to the Loggie of{% endraw %}
{% raw %}Giovanni da Udine (freely but judiciously retouched) on the first{% endraw %}
{% raw %}floor, and those of Raphael on the second (p. 302). The first door{% endraw %}
{% raw %}to the left in the loggie of the first floor leads to the Sistina, and{% endraw %}
{% raw %}that at the end straight before us, to the Museum of Antiquities{% endraw %}
{% raw %}(p. 304).{% endraw %}
{% raw %}Permessi for the Vatican, and Hours of Admission, see pp. 118, 119.{% endraw %}
{% raw %}The visitor should be abundantly provided with 50c. notes and copper{% endraw %}
{% raw %}money, as contributions are exacted from him at about ten different{% endraw %}
{% raw %}places (fee in each department ',2 fr., frequent visitors 25c). The cice¬{% endraw %}
{% raw %}roni who proffer their services at the entrance are generally very ignorant{% endraw %}
{% raw %}and importunate.{% endraw %}
{% raw %}Apartments not usually shown at present, are enclosed within square{% endraw %}
{% raw %}brackets in the following description.{% endraw %}
{% raw %}A. Paintings.{% endraw %}
{% raw %}Cappella Sistina. RaphaeVs Stanze and Loggie. Picture Gallery.{% endraw %}
{% raw %}A permesso may be obtained for any week-day, 8-11 and 2-5 o'clock,{% endraw %}
{% raw %}available for five persons (comp. pp. 118, 119). — The name of each de¬{% endraw %}
{% raw %}partment where it is shown is then deleted in turn.{% endraw %}
{% raw %}We pass the entrance with the Swiss guard and proceed to the{% endraw %}
{% raw %}Scala Rkgia, a magnificent flight of steps, constructed by Antonio{% endraw %}
{% raw %}da Sangallo, and restored by Bernini under Alexander VII., covered{% endraw %}
{% raw %}with tunnel-vaulting resting on Roman columns. We mount these{% endraw %}
{% raw %}steps and pass through a door to the Staircase on the Right, which{% endraw %}
{% raw %}ascends to the first floor, where there is a side entrance of the{% endraw %}
{% raw %}Sistine Chapel (see below), indicated by an inscription, and now{% endraw %}
{% raw %}used by visitors, and to the second floor, where Raphael's stanze{% endraw %}
{% raw %}and loggie are situated (p. 290). The third floor contains the{% endraw %}
{% raw %}picture gallery (p. 296).{% endraw %}
